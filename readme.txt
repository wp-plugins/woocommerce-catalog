Switch to catalog mode with one click (Woocommerce).

== Description ==

This plugin changes your woocommerce e-shop to catalog only shop (without add to cart option).

Some of the features:

1) Catalogue mode with a simple click.
2) Custom button instead of add to cart.
3) Customize button with your colors.
4) Remove price from product loop.


== Installation ==

Upload the plugin to your blog, Activate it, then go to settings page and play with the options.

1, 2, 3: You're done!
